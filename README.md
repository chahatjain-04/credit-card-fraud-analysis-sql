 # Credit Card Transaction Fraud Analysis System

Project Overview

This project simulates a real-world fintech fraud detection system using SQL. It analyzes credit card transactions to identify fraud patterns, high-risk customers, and suspicious activities.

Tech Stack
- SQL (PostgreSQL / MySQL)
- 
Database Design
- 12+ relational tables
- Customers, Cards, Transactions
- Fraud Alerts, Risk Scores, Blacklist
- Devices, Locations, Logins

Key Features
- Fraud detection using transaction patterns
- High-risk customer identification
- Geo & device anomaly detection
- Transaction velocity checks
- Fraud trend analysis

SQL Concepts Used
- Joins
- CTEs
- Window Functions
- Aggregations
- CASE WHEN
- Date functions

Outcome
Built a complete SQL-based fraud detection system showcasing strong data analytics skills in fintech.
